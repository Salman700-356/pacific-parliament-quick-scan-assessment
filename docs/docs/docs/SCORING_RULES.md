# Scoring Rules

## Question scoring
Each question is scored 0–3.
N/A answers are excluded from calculations.

## Pillar scoring
Pillar score = average(score) of all questions in the pillar (excluding N/A)

## Total score
Total score out of 24 = sum of pillar scores across all 8 pillars

## Maturity bands
0–6 Foundational  
6–12 Developing  
12–18 Established  
18–24 Mature
